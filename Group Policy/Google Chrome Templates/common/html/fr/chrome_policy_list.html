ExtensionInstallForcelist

Configurer la liste des extensions dont l'installation est forcée
Type de données :
List of strings
Emplacement dans le registre Windows :
Software\Policies\Google\Chrome\ExtensionInstallForcelist
Nom de préférence Mac/Linux :
ExtensionInstallForcelist
Compatible avec :
  • Google Chrome (Linux, Mac, Windows) depuis la version 9
  • Google Chrome OS (Google Chrome OS) depuis la version 0.11
Fonctionnalités compatibles :
Actualisation dynamique des stratégies: Oui
Description :
Vous permet de définir une liste d'extensions à installer en arrière-plan, sans nécessiter l'interaction de l'utilisateur. Chaque élément de la liste représente une chaîne contenant un identifiant d'extension et une URL de mise à jour, le tout étant délimité par un point-virgule (;). L'identifiant d'extension correspond à la chaîne de 32 caractères que vous pouvez, par exemple, trouver sur chrome://extensions lorsque vous êtes en mode développeur. L'URL de mise à jour redirige vers un document Update Manifest XML, comme cela est décrit sous http://code.google.com/chrome/extensions/autoupdate.html. Pour chaque élément, Google Chrome récupère l'extension définie par l'identifiant d'extension à partir du service de mise à jour sous l'URL de mise à jour spécifiée, puis l'installe en arrière-plan. Par exemple, lcncmkcnkcdbbanbjakcencbaoegdjlp;https://clients2.google.com/service/update2/crx installe l'extension Google SSL Web Search à partir de l'URL de mise à jour standard de Chrome Web Store. Pour plus d'informations sur les extensions hébergées, consultez la page suivante : http://code.google.com/chrome/extensions/hosting.html. Les utilisateurs ne peuvent pas désinstaller les extensions spécifiées par cette règle. Si vous supprimez une extension de cette liste, elle sera désinstallée automatiquement par Google Chrome. Si cette règle n'est pas définie, l'utilisateur peut supprimer n'importe quelle extension dans Google Chrome.
Exemple de valeur :
Windows:
Software\Policies\Google\Chrome\ExtensionInstallForcelist\1 = "lcncmkcnkcdbbanbjakcencbaoegdjlp;https://clients2.google.com/service/update2/crx"
Linux:
["lcncmkcnkcdbbanbjakcencbaoegdjlp;https://clients2.google.com/service/update2/crx"]
Mac:
<array> <string>lcncmkcnkcdbbanbjakcencbaoegdjlp;https://clients2.google.com/service/update2/crx</string> </array>

Nom de préférence Mac/Linux :

  • Google Chrome OS (Google Chrome OS) depuis la version 0.14
Fonctionnalités compatibles :
Actualisation dynamique des stratégies: Non
Description :
Indique le nombre maximal de connexions simultanées au serveur proxy. Certains serveurs proxy ne peuvent pas gérer un grand nombre de connexions simultanées par client. La définition d'une valeur inférieure pour cette règle peut résoudre ce problème. La valeur de cette règle doit être inférieure à 100 et supérieure à 6. La valeur par défaut est 32. Il est déconseillé de définir une valeur inférieure à la valeur par défaut. En cas de non-configuration de cette règle, la valeur par défaut de 32 est utilisée.

URLBlacklist

Bloquer l'accès à une liste d'URL
Type de données :
List of strings
Emplacement dans le registre Windows :
Software\Policies\Google\Chrome\URLBlacklist
Nom de préférence Mac/Linux :
URLBlacklist
Compatible avec :
  • Google Chrome (Linux, Mac, Windows) depuis la version 15
  • Google Chrome OS (Google Chrome OS) depuis la version 0.15
Fonctionnalités compatibles :
Actualisation dynamique des stratégies: Oui
Description :
Bloque l'accès aux URL répertoriées. Cette règle empêche l'utilisateur de charger des pages Web à partir d'URL figurant sur la liste noire. Une URL utilise le format "schéma://hôte:port/chemin". Le schéma facultatif peut être http, https ou ftp. Seul ce schéma sera bloqué. Si aucun n'est spécifié, tous les schémas sont bloqués. L'hôte peut être un nom d'hôte ou une adresse IP. Les sous-domaines d'un nom d'hôte seront également bloqués. Afin d'éviter le blocage des sous-domaines, ajoutez un point (.) avant le nom d'hôte. Le nom d'hôte spécial "*" bloque tous les domaines. Le port facultatif est un numéro de port valide de 1 à 65 535. Si aucun n'est spécifié, tous les ports sont bloqués.
