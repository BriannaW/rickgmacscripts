URLWhitelist

Permitir o acesso a uma lista de URLs
Tipo de dados:
List of strings
Local de registo do Windows:
Software\Policies\Google\Chrome\URLWhitelist
Nome de preferência do Mac/Linux
URLWhitelist
Suportado em:
  • Google Chrome (Linux, Mac, Windows) desde a versão 15
  • Google Chrome OS (Google Chrome OS) desde a versão 0.15
Funcionalidades suportadas:
Atualização Dinâmica de Políticas: Sim
Descrição:
Permite aceder aos URLs listados, enquanto exceções à lista negra de URLs. Consulte a descrição da política de lista negra de URLs para o formato de entradas desta lista. Esta política pode ser utilizada para abrir exceções a listas negras restritivas. Por exemplo, pode incluir-se "*" numa lista negra para bloquear todos os pedidos e esta política pode ser utilizada para permitir o acesso a uma lista limitada de URLs. Pode ser utilizada para abrir exceções para determinados esquemas, subdomínios de outros domínios, portas ou caminhos específicos. O filtro mais específico determinará se um URL é bloqueado ou permitido. A lista de autorizações tem precedência sobre a lista negra. Esta política está limitada a 100 entradas; as entradas subsequentes serão ignoradas. Se esta política não for definida, não haverá exceções à lista negra da política "URLBlacklist".

UserDataDir

Definir diretório de dados do utilizador
Tipo de dados:
String (REG_SZ)
Local de registo do Windows:
Software\Policies\Google\Chrome\UserDataDir
Nome de preferência do Mac/Linux
UserDataDir
Suportado em:
  • Google Chrome (Windows) desde a versão 11
  • Google Chrome (Mac) desde a versão 11
Funcionalidades suportadas:
Atualização Dinâmica de Políticas: Não
Descrição:
Configura o diretório que será utilizado pelo Google Chrome para armazenar dados do utilizador. Se definir esta política, o Google Chrome utilizará o diretório indicado, independentemente de o utilizador ter ou não especificado o sinalizador "--user-data-dir". Se deixar esta política por definir, será utilizado o caminho do perfil predefinido e o utilizador poderá substituí-lo pelo sinalizador de linha de comandos "--user-data-dir".
Valor do exemplo:
"${users}/${user_name}/Chrome"
