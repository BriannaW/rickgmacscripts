  • Google Chrome OS (Google Chrome OS) desde la versión 0.11

Si se habilita o inhabilita esta configuración, los usuarios no podrán cambiarla ni anularla en Google Chrome. Si esta política no se configura, se la habilitará, pero el usuario podrá cambiarla.

ExtensionInstallForcelist

Configurar la lista de extensiones instaladas a la fuerza
Tipo de datos:
List of strings
Ubicación del registro de Windows:
Software\Policies\Google\Chrome\ExtensionInstallForcelist
Nombre de preferencia de Mac/Linux:
ExtensionInstallForcelist
Compatible con:
  • Google Chrome (Linux, Mac, Windows) desde la versión 9
  • Google Chrome OS (Google Chrome OS) desde la versión 0.11
Características admitidas
Actualizar política dinámica: Sí
Descripción:
Permite especificar una lista de extensiones que se podrán instalar automáticamente sin intervención del usuario. Todos los elementos de la lista son cadenas que contienen una ID de extensión y una URL de actualización delimitados por punto y coma (;). La ID de extensión es la cadena de 32 letras que se encuentra, por ejemplo, en chrome://extensions cuando está en modo de desarrollador. La URL de actualización debe apuntar a un documento XML de manifiesto de actualización, tal y como se describe en http://code.google.com/chrome/extensions/autoupdate.html. Para cada elemento, Google Chrome recuperará la extensión especificada por la ID de extensión del servicio de actualización en la URL de actualización especificada y la instalará automáticamente. Por ejemplo, lcncmkcnkcdbbanbjakcencbaoegdjlp;https://clients2.google.com/service/update2/crx instala la extensión Google SSL Web Search de la URL de actualización de Chrome Web Store estándar. Para obtener más información sobre cómo alojar extensiones, consulta http://code.google.com/chrome/extensions/hosting.html. Los usuarios no podrán desinstalar las extensiones especificadas en esta política. Si eliminas una extensión de la lista, Google Chrome la desinstalará de manera automática. Si no se establece esta política, el usuario podrá desinstalar cualquier extensión de Google Chrome.
Valor de ejemplo:
Windows:

La lista blanca tiene prioridad sobre la lista negra. Esta política está limitada a 100 entradas y las entradas posteriores se ignorarán. Si esta política no se configura, no habrá excepciones a la lista negra de la política "URLBlacklist".
